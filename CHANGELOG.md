# 📝 FEED ZUPZUP

## 📌 프로젝트 개요
FEED ZUPZUP은 피드백과 건의사항을 손쉽게 작성하고, 관리자 페이지에서 효율적으로 관리할 수 있는 플랫폼입니다.  
초기 세팅부터 주요 기능(피드백·대시보드·관리자 기능 등) 개발, API 연동, 자동 배포까지 구축한 프로젝트입니다.  

---

## 🚀 v1.0.0 요약

- **프로젝트 초기 세팅**  
  - ESLint, Prettier, StyleLint, Emotion, Storybook, PWA, GA, Sentry, msw 등 개발 환경 구성  
  - GitHub Actions 기반 자동 배포 및 릴리즈 워크플로우 구축  

- **UI/UX 구현**  
  - 기본 레이아웃, 공통 컴포넌트, 헤더·네비게이션  
  - 홈/피드백/건의사항/OnBoarding/대시보드/관리자 페이지 UI 제작  
  - 스타일 변수, 테마 팔레트, CSS reset, 다크/라이트 모드  

- **주요 기능 개발**  
  - 무한 스크롤, 좋아요, 모달, 폼 제출, 태그 필터링  
  - QR 코드 페이지, 방 생성·수정·조회  
  - 로그인/회원가입 및 인증 처리  
  - 관리자 알림 및 피드백 관리 기능  

- **API 연동**  
  - 피드백, 건의사항, 대시보드, 통계  
  - 방/단체 관리, QR 다운로드, 알림, 인증  

- **버그 수정 및 리팩터링**  
  - 무한 스크롤 중복 요청, 피드백 정렬 문제, 대시보드 UI 오류 등 해결  
  - import 경로, 타입 에러, 스타일링 리팩터링  

- **릴리즈 관리**  
  - `v1.0.0` → `v1.1.0` → `v1.1.1` → `v2.0.0`  
  - 시맨틱 버저닝 도입 및 안정적인 배포 환경 구축
